#Проект приложения интернет магазина

Для запуска данного приложения нужно выполнить в терминале команды:
poetry init,
python manage.py runserver
Для перехода по страницам приложения использовать следующие url адреса:
http://127.0.0.1:8000/home/, http://127.0.0.1:8000/contacts/.
Для отправки данных пользователя, необходимо записать имя, телефон, сообщение и нажать кнопку отправить.
После нажатия кнопки отображается обратная связь от сервера об успешной отправке.
